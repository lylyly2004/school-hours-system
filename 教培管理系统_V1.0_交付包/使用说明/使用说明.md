# 教培管理系统 使用说明

版本：V1.0  
适用对象：培训机构、工作室、个体教师  
系统类型：本地版

---

## 一、系统简介

教培管理系统是一套面向小型培训机构的轻量化本地管理工具，适用于日常的：

- 学员报名管理
- 上课记录管理
- 续费与退费管理
- 教师与班级管理
- 财务流水查询
- 数据导出与恢复

系统强调：

- 功能实用
- 操作简洁
- 上手快速
- 本地保存

适合不需要复杂大系统、只希望把学费和课时管理清楚的教培场景。

---

## 二、快速开始

首次使用建议按以下顺序进行：

1. 创建校区  
   位置：`流水管理 -> 校区管理`

2. 创建学科  
   位置：`教学管理 -> 学科管理`

3. 创建教师  
   位置：`教学管理 -> 教师管理`

4. 创建班级  
   位置：`教学管理 -> 班级管理`

5. 创建收费模式  
   位置：`教学管理 -> 收费模式`

6. 绑定数据文件  
   位置：顶部 `数据工具 -> 绑定数据文件`

7. 开始录入真实业务数据  
   推荐顺序：
   - 新生报名
   - 记录上课
   - 续费 / 退费
   - 查询流水

- `index.html`

登录成功后进入后台管理页面。

## 四、日常使用流程

### 1. 新生报名

位置：`前台业务 -> 新生报名`

录入学员报名信息后，系统会自动生成：

- 学员档案
- 报名记录
- 对应报名流水

### 2. 记录上课

位置：`教学管理 -> 上课管理 -> 记录上课`

使用步骤：

1. 选择教师
2. 选择上课日期
3. 选择班级
4. 勾选到课学员
5. 点击 `记录上课`

系统会自动：

- 扣减学员剩余课时
- 记录教师课时
- 生成上课记录

### 3. 查看上课记录

位置：`教学管理 -> 上课管理 -> 上课记录`

可按以下条件查询：

- 日期
- 教师
- 班级

### 4. 学员续费

入口一：

- `首页 -> 查看续费学员`

入口二：

- `教学管理 -> 学员管理 -> 学员详情`

续费后系统会自动：

- 增加课时
- 记录赠课
- 生成续费流水

### 5. 学员退费

位置：

- `教学管理 -> 学员管理`

退费后系统会自动：

- 更新学员状态
- 清零当前剩余课时
- 保留历史上课记录
- 生成退费流水

## 五、数据保存说明

### 1. 本地自动保存

系统会自动保存当前录入和修改的数据。  
正常使用时，不需要每一步手动保存。

### 2. 绑定数据文件

顶部入口：

- `数据工具 -> 绑定数据文件`

建议首次使用时绑定一个固定的本地 `json` 数据文件，用于：

- 长期保存数据
- 后续恢复数据
- 换电脑迁移数据

### 3. 从文件同步

顶部入口：

- `数据工具 -> 从文件同步`

当页面数据异常，或需要重新从绑定文件读取数据时，可使用此功能。

## 六、备份与恢复

### 1. 导出数据

顶部入口：

- `数据工具 -> 导出数据`

建议：

- 每次录入一批重要数据后导出一次
- 每周至少保留一次完整备份

### 2. 导入数据

顶部入口：

- `数据工具 -> 导入数据`

可将之前导出的 `json` 数据文件重新导入系统。

### 3. 换电脑迁移

如果后续更换电脑，可按以下方式迁移：

1. 在旧电脑点击 `导出数据`
2. 将导出的 `json` 文件复制到新电脑
3. 在新电脑打开系统
4. 点击 `导入数据`
5. 选择之前导出的文件完成恢复

## 七、常见问题

### 1. 为什么学员剩余课时会出现负数？

系统支持学员在欠费情况下继续上课。  
例如剩余课时为 `0` 时继续上课，会显示：

- `-1`
- `-2`

续费后会自动抵扣负数，再继续按正常课时计算。

### 2. 为什么某些班级、收费模式、校区不能删除？

为了保证历史数据安全，系统会限制删除正在使用中的基础数据。  
例如：

- 班级下还有在读或停课学员
- 收费模式仍有学员正在使用
- 校区已有学员或流水记录

这种情况下，需要先处理关联数据，才可删除。

### 3. 如果页面数据看起来不对怎么办？

建议按以下顺序处理：

1. 暂停继续录入新数据
2. 点击 `数据工具 -> 从文件同步`
3. 如果仍异常，再使用之前导出的备份文件执行 `导入数据`

## 八、使用建议

为了保证系统长期稳定使用，建议：

- 定期执行 `导出数据`
- 修改重要信息前，确认当前数据已正常保存
- 不要随意删除正在使用中的班级、收费模式、校区等基础数据
- 系统升级或修改后，先进行小范围测试再继续录入正式数据
