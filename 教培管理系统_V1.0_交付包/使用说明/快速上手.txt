教培管理系统 快速上手

一、打开系统
1. 使用浏览器打开 `index.html`
2. 输入交付时提供的账号密码登录

二、首次使用建议
首次使用请先完成以下基础设置：
1. 流水管理 -> 校区管理
2. 教学管理 -> 学科管理
3. 教学管理 -> 教师管理
4. 教学管理 -> 班级管理
5. 教学管理 -> 收费模式

三、建议先做的数据保存操作
1. 顶部 `数据工具 -> 绑定数据文件`
2. 绑定一个固定的本地 json 数据文件
3. 以后定期执行 `数据工具 -> 导出数据`

四、日常使用顺序
1. 前台业务 -> 新生报名
2. 教学管理 -> 上课管理 -> 记录上课
3. 首页或学员管理中查看续费学员
4. 财务管理 -> 流水管理 查询记录

五、如果换电脑
1. 旧电脑点 `导出数据`
2. 把导出的 json 文件复制到新电脑
3. 新电脑打开系统后点 `导入数据`

六、如果页面数据异常
1. 先不要继续录入新数据
2. 点 `数据工具 -> 从文件同步`
3. 如仍异常，再使用之前导出的备份执行 `导入数据`

七、建议
1. 每录入一批重要数据后，导出一次备份
2. 修改系统前，先确认数据已经导出
3. 不要随意删除正在使用中的校区、班级、收费模式
